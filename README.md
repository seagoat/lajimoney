# 可转债折溢价套利系统

可转债折溢价套利量化模拟系统，支持持仓对冲模式和全市场裸套模式。

---

## 核心原理

### 什么是可转债

可转债（Convertible Bond，CB）是上市公司发行的债券，持有人可以在约定时间内按约定价格将债券转换为公司股票。

### 折溢价套利基本逻辑

可转债具有"债性"和"股性"双重属性：
- **债性**：作为债券持有，到期还本付息
- **股性**：可以按转股价转换为正股

**转股价值** = 正股价格 ÷ 转股价 × 100

例如：正股价格 10 元，转股价 8 元，则每张转债的转股价值 = 10 ÷ 8 × 100 = 125 元。

**溢价率** = （转债价格 - 转股价值）÷ 转股价值 × 100%

| 情况 | 含义 |
|------|------|
| 溢价率为正 | 转债价格 > 转股价值，转债偏贵 |
| 溢价率为负（折价） | 转债价格 < 转股价值，存在套利空间 |

**折价（Discount Space）** = 转股价值 - 转债价格

当折价存在时，买入转债 → 立即转股 → 卖出正股，可以获得价差收益。

### 持仓对冲模式（HEDGE）

用户持有正股底仓时的套利方式：

1. 收盘前扫描：检查持仓正股对应的转债是否出现**折价**（溢价率 < 阈值，如 -1%）
2. 买入转债：买入张数根据底仓股数自动换算（见下）
3. 立即转股：将持有的转债按转股价转换为正股
4. 卖出正股：按收盘价卖出持有的正股

**买入张数换算**：系统按底仓股数自动计算所需转债张数
```
所需张数 = ceil(持有股数 × 转股价 ÷ 100 ÷ 10) × 10
```
例如：持有 1000 股，转股价 10 元 → 需买入 100 张 CB（10 手）

**对冲效果**：买转债 + 转股 + 卖正股，净持仓不变，锁定价差收益。

### 裸套模式（NAKED）

用户没有正股底仓时的套利方式：

1. 全市场扫描：寻找折价较大的转债（溢价率 < 裸套阈值，如 -2%，需要更大的安全垫）
2. 买入转债 + 立即转股
3. 次日开盘卖出正股

**关键假设**：正股次日开盘价不会大跌超过安全垫（如 2%），否则转股后卖出会亏损。

**风险**：如果正股次日大跌，转股卖出亏损。2% 折价作为安全垫需要次日正股跌幅 < 2% 才能盈利。

裸套模式需要更大的折价阈值作为风险补偿。

## 关键公式

```
转股价值 = 正股价格 ÷ 转股价 × 100
溢价率   = (转债价格 ÷ 转股价值 - 1) × 100%
折价空间 = 转股价值 - 转债价格
```

### 持仓对冲收益计算

```
卖出总额  = 卖出价 × 100 × 张数
印花税    = 卖出总额 × 印花税率（千1.5，仅卖出）
券商佣金  = 卖出总额 × 券商佣金率（万1，买卖均收）
转债买入费 = 买入价 × 100 × 张数 × (1 + 佣金率)
净利润    = 卖出总额 - 印花税 - 券商佣金 - 转债买入费
收益率    = 净利润 ÷ 转债买入费 × 100%
```

### 裸套模式收益计算

```
CB买入成本  = 买入价 × 100 × 张数 × (1 + 佣金率)
转股获得股数 = 100 ÷ 转股价 × 买入张数
次日卖出入账 = 次日开盘价 × 股数 × (1 - 佣金率)
净利润      = 卖出入账 - CB买入成本
收益率      = 净利润 ÷ CB买入成本 × 100%
```

## 系统功能

### 底仓管理
在"底仓管理"页面添加持有的正股（系统只记录，用户自行在券商账户操作）

### 手动扫描
首页点击"手动触发扫描"：
- **持仓模式**：仅扫描底仓正股对应的转债，速度快
- **全市场模式**：扫描全市场所有转债（耗时约 20-60 秒），底仓为空时自动启用裸套模式

### 交易成本配置
在"扫描设置"页面可配置：
- 股票券商佣金率（默认万1）
- 股票印花税率（默认千1.5，仅卖出）
- 转债券商佣金率（默认万0.6）

### 成交记录与结算
- 系统自动记录模拟买入，扫描结果实时显示买入价、正股价、估算交易费
- 手动结算：按次日模拟开盘价（=买入价 × 1.001）计算收益
- 收益含完整交易成本

## 核心参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 折价阈值 | -1.0% | 溢价率低于此值触发信号（持仓模式） |
| 每次买入 | 自动换算 | 按底仓股数自动计算（见持仓对冲模式说明） |
| 扫描模式 | 仅持仓正股 | 或全市场转债 |
| 裸套折价阈值 | -2.0% | 裸套专用，要求更严格的折价 |
| 股票佣金 | 万1（0.01%） | 买卖均收 |
| 印花税 | 千1.5（0.15%） | 仅卖出时收取 |
| 转债佣金 | 万0.6（0.006%） | 买卖均收 |

## 系统要求

- Python 3.10+
- Windows / macOS / Linux
- 网络连接（实时价格请求）

## 快速启动

### Windows

```batch
.\run.bat
```

### macOS / Linux

```bash
./run.sh
```

浏览器访问：http://localhost:8001

## 技术架构

- **后端**：FastAPI + SQLite (aiosqlite)
- **前端**：Jinja2 模板 + 原生 JavaScript
- **数据源**：新浪财经（实时价格）+ THS 同花顺（转债基础信息）
- **环境管理**：uv（首次运行自动安装）
